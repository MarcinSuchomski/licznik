# Witam w aplikacji licznik!

Backend Laravel 

Frondend Vue.js

DB sqLite

Aby uruchomic aplickacje:
* copy `database.sqlite` z `setup_files` do 'licznik\database'
*  run `npm install`
*  run `npm install vue-router vue-axios`
*  run `npm install`


## Useful commands

 * `npm run build`   compile typescript to js
